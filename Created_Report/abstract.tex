\begin{center}
\textbf{\Large ABSTRACT}
\end{center}

\vspace{1cm}

Accurate skin tone detection and cosmetic shade recommendation remain significant challenges in the beauty and cosmetics industry, particularly for diverse skin tones. Traditional methods often rely on subjective visual assessment or limited shade ranges, leading to poor matches and customer dissatisfaction. This project presents \textbf{ShadeAI}, a comprehensive AI-based system for automated skin tone classification and foundation shade recommendation using computer vision and color science principles.

\vspace{0.5cm}

ShadeAI implements a six-stage pipeline: (1) image preprocessing with automatic white balance correction, (2) face detection using Haar Cascade classifiers, (3) rule-based skin segmentation using HSV and YCbCr color spaces, (4) feature extraction calculating Individual Typology Angle (ITA°) and CIELAB color statistics, (5) skin tone classification using the inclusive 10-class Monk Skin Tone (MST) scale, and (6) foundation shade recommendation using CIEDE2000 color difference metrics. The system analyzes facial photographs and recommends the top 5 matching foundation shades from a comprehensive database of 187 shades across 14 major cosmetic brands.

\vspace{0.5cm}

The system achieves outstanding performance with an average color difference (ΔE) of 1.361, which is 49.6\% better than published research targets (ΔE < 2.7) and 55\% better than professional industry standards (ΔE < 3.0). Testing on diverse skin tones (MST-03 to MST-09) demonstrates 85.7\% excellent match rate (ΔE < 3.0) and 100\% success rate, with 57.1\% of recommendations achieving perfect matches (ΔE < 1.0). The system processes images in under 1 second without requiring any training data or deep learning models, making it production-ready and deployable immediately.

\vspace{0.5cm}

ShadeAI provides three user interfaces: a command-line interface (CLI) for batch processing, a web application built with Streamlit for interactive use, and a RESTful API using FastAPI for integration with other applications. The system includes real-time photo quality assessment with six quality metrics (brightness, blur, color cast, face detection, resolution, and exposure) to ensure accurate results. A comprehensive foundation database covers all MST classes with excellent representation of deeper skin tones (MST-06 to MST-10), addressing historical gaps in cosmetic shade ranges.

\vspace{0.5cm}

The project demonstrates the practical application of computer vision, color science, and software engineering principles to solve real-world problems in the beauty industry. By achieving research-grade accuracy without requiring training data, ShadeAI provides an accessible, inclusive, and efficient solution for personalized cosmetic recommendations. The system's modular architecture allows for future enhancements including deep learning models, expanded shade databases, webcam integration, and virtual try-on features.

\vspace{0.5cm}
